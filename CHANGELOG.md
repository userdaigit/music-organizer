# 变更日志

## v1.1.0 (2026-07-06)

### 新功能
- **进度条显示**：所有耗时步骤（扫描、元数据提取、歌手规范化、网络刮削、指纹识别、复制文件）均显示 pip 风格百分比进度条
- **输出目录自动创建**：输出目录不存在时自动创建，权限不足时提示手动创建
- **源目录检查**：源目录不存在时报错退出，避免无效运行
- **版本号集中管理**：新增 `version.py`，所有文件统一引用，避免散落多处
- **Windows 启动脚本**：新增 `organize.ps1`，支持 PowerShell 环境一键启动

### 工程改进
- **正式测试套件**：新增 `tests/` 目录，106 个 pytest 测试覆盖核心逻辑（feat.解析、编码修复、进度条、路径构建等）
- **CI/CD**：新增 GitHub Actions 配置，Python 3.9/3.11/3.13 矩阵自动测试
- **分发自动化**：新增 `sync_dirs.ps1`，自动同步 github-release 到 script/docker 目录
- **歌手规范化 API**：新增 `normalize_one()` 方法，支持逐个显示进度

### 文档更新
- 命令行参数和帮助信息增加路径替换提示
- README 中 User-Agent 修改位置更新为引用 version.py
